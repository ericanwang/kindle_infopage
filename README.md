# kindle_infopage
为kindle设计的一个页面，使用kindle的浏览器访问该页面能够显示时间、日期。后期考虑加入农历和天气等其它功能。
## 禁止 Kindle 屏幕自动休眠和锁屏
在Kindle 的搜索框中输入如下代码并回车，就可以禁止 Kindle 自动锁屏，让屏幕常亮：~ds「~ds」是「Disable Screensaver」的缩写。输入该代码后，即使手动按下 Kindle 的电源键，Kindle 也不会锁屏。
